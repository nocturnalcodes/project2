Project Concept

This website is all about the 1990s Chicago Bulls! Called 'the greatest NBA team of all time',, this will be a blog-style, informative website with the intended audience being basketball fans. The objectives of this website are:

  1. To educate and inform basketball fans of all ages about 90s Chicago Bulls!
  2. Serve as a time capsule for fans to relive the memories and impact this team has had on the NBA.
  3. Provide detailed information to educate new school basketball fans who don't know much about how fascinating this team was.
  4. Incorporate engaging visual elements such as pictures, video clips, and an interactive timeline that provides additional insights to the history of the Chicago Bulls.

Key Features:

  • Interactive Timeline
  • Short Clips of Game Recaps
  • Detailed Player Profiles
  • Virtual Museum

The page layout I will use is as follows: The website will follow a 'Z-Pattern'. A 'Z-Pattern' describes a design layout that guides the viewers' eyes in the shape of a Z. The 'Z-Pattern' layout can be seen in more detail following this link.
The website will feature a navigation menu at the top of each page to help visitors find and navagate each webpage.

Project Resource:

Youtube

Youtube is an excellent resource for learning topics in Web Design. Since this field is visual-oriented, seeing a video demonstration of web design principles can be really helpful and sometimes more impactful than written resources. Youtube has videos about principles such as visual hierarchy, contrast, color, repetition, and alignment. Youtube also has videos that showcases the process of designing a website from start to finish.

I will use Youtube in my development process to get inspiration for the layout of my website. I will also use Youtube to find creators who show how they design their websites using HTML and CSS. I sometimes feel like I am limitied in my ability to make a modern website using those 2 coding languages, but channels like this one will help me create a well-rounded website!
